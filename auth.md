# Authentication — Primus Digital

This site is fully public. There are no protected APIs, no user accounts, no
login, and no agent registration requirement. Agents do not need credentials
to read any resource on this domain.

- All content is available anonymously over HTTPS.
- A markdown version of the site is served at `/index.md` (or request `/` with
  `Accept: text/markdown`).
- Machine-readable resources are listed at `/.well-known/api-catalog` and
  `/.well-known/agent-skills/index.json`.
